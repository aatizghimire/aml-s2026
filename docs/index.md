## Course Description

### Overview
**Adversarial Machine Learning (AI Security)** is a comprehensive 10-week hands-on course that teaches you how to attack and defend machine learning systems. You'll learn to think like an AI red teamer — identifying vulnerabilities in ML systems, executing sophisticated attacks, and implementing effective defenses.

From image classifiers to large language models, you'll master the full spectrum of adversarial machine learning techniques used by security professionals and researchers worldwide.

This course bridges the gap between theoretical ML security research and practical offensive operations. You won't just read about attacks — you'll write them from scratch, test them against real models, and build the skills to assess ML systems in any environment.

### Why This Course Matters
Machine learning is everywhere — from email spam filters and facial recognition to malware detectors and LLM-powered chatbots. But as ML adoption grows, so do the attack surfaces.

Real attacks are happening right now:

* Researchers extracted training data from ChatGPT, revealing phone numbers and email addresses
* Prompt injection attacks have compromised LLM-powered applications across major tech companies
* Model extraction attacks have stolen intellectual property from paid API endpoints
* Adversarial examples have fooled self-driving car systems and facial recognition

> **The problem:** Most security professionals don't understand ML vulnerabilities. Most ML engineers don't understand security. This course makes you the bridge.
